# DramaVocab 3.0 对外介绍文案

## 一句话介绍
DramaVocab（抓马英语）是一个把“爽文追更体验”和“英语词汇学习”融合在一起的阅读学习产品。

## 30 秒介绍（适合发朋友圈/社群）
看最狗血的文，背最高级的词。  
DramaVocab 让你在同一段剧情里一键切换三种学习强度：沉浸读、透视看释义、挑战填空。  
错词自动进复习，不用手动整理；还支持导入你自己的小说，变成可学习版本。  
如果你背单词总坚持不住，来试试“先上头，再记住”。

## 长版介绍（适合 README / 社区帖子）
DramaVocab 的目标是把“背单词”从痛苦任务变成连续剧情体验。  
产品内置系统书籍，并支持用户快速导入私藏文本，自动进入三模式阅读链路：
- 沉浸模式：先读懂剧情，保持阅读快感
- 透视模式：词义直出，减少跳转查词
- 挑战模式：填空检验记忆，错词自动沉淀复习

我们优先保证：页面丝滑、链路完整、学习动作可持续。  
一句话：不是词表工具，而是剧情驱动的词汇学习体验。
